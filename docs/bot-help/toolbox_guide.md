# 工具箱

如果想要上传并查询suite和mysekai数据，请先前往https://haruki.seiunx.com，点击左下角选择注册。

## 游戏UID绑定

注册成功登陆后，点击左下角后选择游戏账号管理，然后选择绑定新账号、选择你游戏账号对应区服并在游戏打开个人信息页复制游戏UID，随后点击验证点击屏幕中心的数字组合进行复制，并在游戏中个人信息界面修改签名为对应数字组合并在游戏中返回主页面来保存签名，确认签名已修改后点击工具箱右下角“我已输入，关闭此窗口”按钮。

<CollapseBox title=数据设置参数说明>

- 允许公开api访问：一般无需打开，除非你使用的服务说明需要打开此选项，打开后你的数据将能被任何人直接读取

</CollapseBox>

## 社交平台账号绑定

进入工具箱登陆后点击左下角选择账号设置，向下翻阅找到社交平台账号绑定设置，并输入你的qq账号点击发送邮件，填入验证码后点击验证即可。
> [!caution]
> 请务必确保你的qq邮箱已启用并且可以收到邮件

## QQ官方机器人”宵崎奏“绑定
> [!caution]
> 由于平台限制，QQ官方机器人“宵崎奏”在使用需要suite/mysekai数据的功能时无法读取你在一般HarukiBot分布式中的绑定信息  
> 因此在使用“宵崎奏”Bot前您需要重新执行QQ官方Bot的绑定

### 第一步: 登陆Haruki工具箱
登陆[Haruki工具箱](https://haruki.seiunx.com/)，进入[账号设置](https://haruki.seiunx.com/user/settings)
找到`授权社交平台查询`，并点击其中的`新增授权`，再点击`平台-QQ官方Bot`
![授权社交平台查询](..\assets\qqofficial-guide\1-1.webp)
![新增授权](..\assets\qqofficial-guide\1-2.webp)
### 第二步: 获取用户ID
在加入了"宵崎奏"Bot的群内，发送消息`@宵崎奏 /查qid`
你将会看到一个类似如下的图片
![获取用户ID](..\assets\qqofficial-guide\2-1.webp)
其中的`QID`就是你的用户ID，请全部复制
### 第三步: 进行工具箱的绑定操作
在复制完用户ID后，回到Haruki工具箱的绑定页面，将`账号`填写为刚刚复制的ID，并点击`保存`
![绑定](..\assets\qqofficial-guide\3-1.webp)
绑定成功后，你可以在Haruki工具箱的`授权社交平台查询`中查看到你的绑定信息
### 关于快速验证
如上图的`新增社交平台授权`中所示，有一个`允许快速验证`功能  
开启了该功能之后，该授权账号即可在HarukiBot NEO中**快速通过账号所属权验证**  
因此强烈建议只允许自己的账号启用该功能
