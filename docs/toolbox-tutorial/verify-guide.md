# Haruki工具箱 - 快速验证账号教程
> [!caution]
> 为保护用户部分账号相关功能不被他人滥用，需要前往Haruki工具箱绑定社交平台账号并验证之后才能正常使用相关功能

登陆[Haruki工具箱](https://haruki.seiunx.com/)，进入[账号设置](https://haruki.seiunx.com/user/settings)
找到`授权社交平台查询`，并点击其中的`新增授权`，选择`QQ`并输入自己想要验证的账号，最后<span style="color:red">打开“允许快速验证”选项   </span> 并保存
![授权社交平台查询](..\assets\qqofficial-guide\1-1.webp)
![新增授权](..\assets\qqofficial-guide\1-2.webp)
开启了该功能之后，该授权账号即可在HarukiBot NEO中**快速通过账号所属权验证**  
因此强烈建议只允许自己的账号启用该功能
